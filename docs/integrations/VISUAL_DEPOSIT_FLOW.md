### 1. **Webhook is the Trigger**

- NowPayments sends webhook when Bitcoin is confirmed
- Webhook contains `payment_status: "finished"`

- Webhook signature must be verified
- Prevents fake deposits
- Uses HMAC-SHA512 encryption


- Polls API every 30 seconds
- Automatic refresh after actions
- Real-time updates
